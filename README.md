
# Fullstack Symfony 6 project 




## Install dependencies

```bash
  composer install
```


## Create and hydrate Database

```bash
  php bin/console d:d:c
  php bin/console d:ma:mi
  php bin/console d:mi:mi
  php bin/console d:f:l -n
```

Start the server

```bash
  symfony serve
```
